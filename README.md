# 🤖 Basic Rule-Based Chatbot

## Description

This project is a simple Rule-Based Chatbot developed using Python. It responds to predefined user inputs such as greetings, basic conversation, programming-related questions, and other common queries. The chatbot uses dictionaries, loops, functions, and conditional statements to generate responses.

---

## Features

- Greeting messages
- Basic conversation
- Programming-related questions
- Predefined responses
- Continuous chat until the user enters `bye`
- Simple and easy-to-understand code

---

## Technologies Used

- Python 3
- Dictionary
- Functions
- Loops
- If-Else Statements

## How to Run

1. Open the project folder.
2. Open Terminal or Command Prompt.
3. Run the following command:

```bash
python main.py
```

## Sample Conversation

```
You : hello
Bot : Hello! How can I help you?

You : how are you
Bot : I'm doing well. Thanks for asking!

You : what is python
Bot : Python is a high-level programming language.

You : thank you
Bot : You're welcome!

You : bye
Bot : Goodbye! Have a great day.
```
